This CD holds symbols denoting various types of numbers, vectors and similar constructs. Principally these are required for compatibility with MathML.
Concepttype
A symbol to be used within an OpenMath attribution to convey the type of the object.
Example

The variable z is attributed with a type for complex numbers.

zcomplex_cartesian_type

Conceptconstant_type
A type for the common constants, pi ~= 3.1415, e ~= 2.718, i = square root of -1, gamma ~= .5772, NaN, infinity (all in the nums cd), true and false (in the logic cd). Also for MathML variables declared to have type constant, as in <ci type="constant">x</ci>.
