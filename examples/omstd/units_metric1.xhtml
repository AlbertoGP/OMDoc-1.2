The OpenMath Content Dictionary units_metric1.ocd in OMDoc form
Michael Kohlhase, The OpenMath Society
This CD defines symbols to represent the basic physical units.
Conceptmetre
This symbol represents the measure of one metre. This is the standard metric unit measure for physical distance.
Type(sts.omdoc#sts)
length
Conceptmetre_sqrd
This symbol represents the measure of one metre squared. This is the standard metric measure for physical area.
Conceptlitre
This symbol represents the measure of one litre. This is the standard metric measure for physical volume.
Conceptmetres_per_second
This symbol represents the measure of one metre per second. This is the standard metric measure for speed.
Conceptmetres_per_second_sqrd
This symbol represents the measure of one metre per second squared. This is the standard metric measure for acceleration.
Conceptsecond
This symbol represents the measure of one second. This is the standard metric measure for time.
Type(sts.omdoc#sts)
time
Conceptgramme
This symbol represents the measure of one gramme. This is the standard metric measure for mass.
Type(sts.omdoc#sts)
mass
ConceptNewton
This symbol represents the measure of one Newton. This is the standard metric measure for force.
Type(sts.omdoc#sts)
force
Conceptdegree_Kelvin
This symbol represents the measure of one degree Kelvin. This is a standard metric measure for temperature relative to absolute zero.
Type(sts.omdoc#sts)
temperature
Conceptdegree_Celsius
This symbol represents the measure of one degree Celsius. This is a standard metric measure for temperature.
Type(sts.omdoc#sts)
temperature
ConceptNewton_per_sqr_metre
This symbol represents the measure of one Newton per square metre. This is a standard metric measure for pressure.
Type(sts.omdoc#sts)
pressure
ConceptCoulomb
This symbol represents the measure of one Coulomb. This is the standard metric measure for charge.
Type(sts.omdoc#sts)
charge
Conceptamp
This symbol represents the measure of one amp. This is the standard metric measure for current.
Type(sts.omdoc#sts)
current
Conceptvolt
This symbol represents the measure of one volt. This is the standard metric measure for voltage.
Type(sts.omdoc#sts)
voltage
