ConceptIndType
Constructor for Inductive Types. Takes arguments the constructor functions for the inhabitants of the type and their signatures.
Example

This is the defining mathematical property for the symbolic type nat of the inductively defined natural numbers with 0 and succ as constructor functions.

IndType( 0 ℕ,sℕ↦ℕ) 0
