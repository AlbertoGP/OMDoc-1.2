Conceptdiff
This symbol is used to express ordinary differentiation of a unary function. The single argument is the unary function.
Type(sts.omdoc#sts)
Object↦Object↦Object↦Object
Lemma

diff(lambda y:a(y) + b(y))(x) = diff(lambda y:a(y))(x) + diff(lambda y:b(y))(x)

λya(y)+b(y)′(x)=λya(y)′(x)+λyb(y)′(x)
Lemma

λya(y)⁢b(y)′(x)=λya(y)′(x)⁢b(x)+a(x)⁢λyb(y)′(x)

λya(y)⁢b(y)′(x)=λya(y)′(x)⁢b(x)+a(x)⁢λyb(y)′(x)
Example

λxx+1.0′(y)=1.0 for all y.

λxx+1.0′(y)=1.0

Conceptint
This symbol is used to represent indefinite integration of unary functions. The argument is the unary function.
Type(sts.omdoc#sts)
Object↦Object↦Object↦Object
Lemma

Application of integration followed by differentiation is the identity function, that is: λxλy∫f(z)ⅆz(y)′(x)=λxf(x).

λxλy∫f(z)ⅆz(y)′(x)=λxf(x)
Example

An example which represents the equation: ∫sinxⅆx=λx-cosx.

∫sinxⅆx=λx-cosx
Conceptdefint
This symbol is used to represent definite integration of unary functions. It takes two arguments; the first being the range (e.g. a set) of integration, and the second the function.
Type(sts.omdoc#sts)
RangeOfIntegration,Object↦Object↦Object
Lemma

for all a,b ∫a,bf(x)ⅆx=-∫b,af(x)ⅆx.

∀a,b.∫a,bf(x)ⅆx=-∫b,af(x)ⅆx
Lemma

For all a, b, c with a<bb<c we have ∫a,cf(x)ⅆx=∫a,bf(x)ⅆx+∫b,cf(x)ⅆx.

∀a,b,c.a<b∧b<c∧∫a,cf(x)ⅆx=∫a,bf(x)ⅆx+∫b,cf(x)ⅆx
Example

An example to represent the definite integration of sin(x) between the points -1.0 and 1.0.

∫-1.0,1.0sinxⅆx
Example

An example to represent the definite integration of f(x), for x in the set C:

∫Cf(x)ⅆx
