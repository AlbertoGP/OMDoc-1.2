Michael Kohlhase, The OpenMath Society
This CD defines symbols to represent imperial standard measures.
Conceptmile
This symbol represents the measure of one mile. This is the standard imperial measure for distance.
Type(sts.omdoc#sts)
length
Conceptacre
This symbol represents the measure of one acre. This is the standard imperial measure for area.
Type(sts.omdoc#sts)
area
Conceptpint
This symbol represents the measure of one pint. This is the standard imperial measure for volume.
Type(sts.omdoc#sts)
volume
Conceptmiles_per_hr
This symbol represents the measure of one mile per hour. This is the standard imperial measure for speed.
Type(sts.omdoc#sts)
velocity
Conceptmiles_per_hr_sqrd
This symbol represents the measure of one mile per hour squared. This is the standard imperial measure for acceleration.
Type(sts.omdoc#sts)
acceleration
Conceptpound_mass
This symbol represents the measure of the mass which weighs one pound under the influence of standard gravity.
Type(sts.omdoc#sts)
mass
Conceptpound_force
This symbol represents the measure of force of one pound.
Type(sts.omdoc#sts)
force
Conceptdegree_Fahrenheit
This symbol represents the measure of one degree Fahrenheit. This is the standard imperial measure for temperature.
Type(sts.omdoc#sts)
temperature
Conceptbar
This symbol represents the measure of one bar. This is the standard imperial measure for pressure.
Type(sts.omdoc#sts)
pressure
