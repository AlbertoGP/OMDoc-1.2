This CD holds the definitions of the unique existential ("there exists exactly one") quantifier and the definite description operator.
Conceptexists-unique
This symbol represents the unique existential ("there exists exactly one") quantifier which takes two arguments. It must be placed within an OMBIND element. The first argument is the bound variables (placed within an OMBVAR element), and the second is an expression.
Type(sts)
binder
Conceptthat
This symbol represents the definite description operator, which takes two arguments. It must be placed within an OMBIND element. The first argument is the bound variables (placed within an OMBVAR element), and the second is an expression. If there is a unique set of variable value assignments such that the expression is true, the operation evaluates to this set of values; otherwise, the result will be undefined.
Type(sts)
binder
