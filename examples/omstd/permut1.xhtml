Conceptpermutation
The n-ary constructor permutation. The arguments are the integers 1 .. n in some order. permutation(p1, ..., pn) is the function which takes 1 to p1, 2 to p2 and so on.
Type(sts.omdoc#sts)
PositiveInteger*↦Permutation
