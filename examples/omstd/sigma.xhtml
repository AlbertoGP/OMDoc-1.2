The OpenMath Content Dictionary sigma.ocd in OMDoc form
Michael Kohlhase, Olga Caprotti
Sigma Types for OM Declaration of symbols for types and objects An OpenMath object which uses any of the symbols here defined, has to use them in accordance with the definitions in ESPRIT deliverable 1.3.2b .
ConceptSigmaType
The type constructor of cartesian products. It takes a list of type-attributed variables and an OpenMath object.
ConceptPair
The pairing constructor. It takes two OpenMath objects as first element and second element of the pair, and a third optional OpenMath object that represents the type of this pair.
ConceptPairProj1
The first projection function. It satisfies sigma-reduction.
ConceptPairProj2
The second projection function. It satisfies sigma-reduction.
