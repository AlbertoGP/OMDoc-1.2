The direct product $\varphi $ is a composition $\varphi : \mathop \times \limits _{i=1}^n F_i \times \mathop \times \limits _{i=1}^n F_i \rightarrow \mathop \times \limits _{i=1}^n F_i$ It can be defined componentwise within every semigroup by: $f_i : F_i \times F_i \rightarrow F_i$ f"ur $i = 1, ..., n$ If $(x_1, ..., x_n), (y_1, ..., y_n) \in \mathop \times \limits _{i=1}^n F_i$, then is $\varphi ((x_1, ..., x_n), (y_1, ..., y_n)) = = (f_1(x_1,y_1), ..., f_n(x_n,y_n)) = (z_1, ..., z_n)$ That means $\forall i \in set(1, ..., n)$: we have $z_i = f_i (x_i, y_i)$.
